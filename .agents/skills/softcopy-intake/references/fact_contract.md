# Fact Contract

Core required facts are defined only in `softcopy/contracts/required_facts.yaml`.

Rules:

1. Formal readiness accepts only `confirmed` core facts.
2. `software_name_short` is optional.
3. `first_publication_status` is required for formal readiness and must be confirmed.
4. `first_publication_date` becomes required only when publication status is confirmed as `published`.
